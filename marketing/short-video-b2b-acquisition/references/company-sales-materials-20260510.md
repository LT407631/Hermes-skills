## 一、成交转化.pptx — 成交体系

### 成交四阶段

```
破冰 → 基础信任 → 消除信息差 → 临门一脚
```

### 成交八要素（必须刻在脑子里）

```
基础信任 + 个人价值 + 品质认可 + 需求满足 + 唯一性 + 担忧解除 + 紧迫性 + 得到便宜 = 成交
```

### 破冰与信任

- 客户为什么会防御？为什么会抗拒？
- 破冰四要素：身份、成就、气质、交谈（形象、礼仪、道具）
- **解除抗拒公式：** 好奇吸引 → 理念同频 → 焦点转移
- **信任核心理念：** 帮客户规避风险和损失
- 高手谈单 80% 的时间，都是在解决信任问题

### 紧迫性

- 紧迫性 =/= 逼单。逼单往往没用
- 紧迫性是过程中塑造的，不是最后给的
- **四张牌：** 爽点、痛点、竞争、稀缺

### 个人价值

- **专业价值：** 听懂客户的话→能帮到他→能解决问题减缓焦虑
- **资源价值：** 省钱、省事

### 客户常见问题 24 问

1. 你们都有什么板子？
2. 你们板子有授权吗？
3. 你们工厂在哪里？
4. 封边用什么？柜门、柜体、背板都是 PUR？
5. 你们五金用什么品牌？什么系列的？
6. 2 米以上的柜门一般用几个铰链？
7. 顶封板、收口条和踢脚板要不要加钱？
8. 一门到顶是否要加钱？
9. 抽屉超过多少钱一个？抽屉无论大小一致价格？
10. 衣柜里面的挂衣杠有配吗？
11. 斜切拉手等免拉手设计收费呢？
12. 自购一些功能五金能帮忙免费安装吗？
13. 如果有异形，需要异形费吗？
14. 柜门最多能做多高？
15. 有需要收费的色号吗？
16. 拉直器怎么算钱？灯带多少 1 米？
17. 柜门柜体背板厚度多少？
18. 抽屉材质与柜门柜体一致？整体厚度是多少？底板厚度是多少？
19. 所有的五金包括铰链、导轨能自购你们安装吗？
20. 抽屉轨道标配侧装三节轨还是托底轨？
21. 背景板、单板是什么价？

23. 合同怎么付款？
24. 工期多久？

## 二、怎么快速提升你的邀约率.pptx — 风格话术库

### 核心洞察

> **「我们凭什么跟小工厂竞争？人家的设计、产品、服务真的差吗？你如果卖跟本地工厂一样的产品（精板），你的邀约率上不来，难道不是理所应当的吗？」**

### 未到店客户聊天万能公式

```
认可人 → 认可理念 → 理念同频 → 焦点转移（问句结尾）
```

**举例（客户问「你们有没有万华禾香板」）：**
1. 认可人：「您看来是对全屋定制的板材有了比较深的了解了，您是想用这款板子做定制吗？」
2. 认可理念：「您想选择这款板子一定是因为环保标准比较高可以达到 ENF 级，使用了无醛 MDI 胶，是这样吗？」
3. 理念同频：「我们工厂和您一样也特别注重环保，因为我们之前是做出口的，我们使用的板材除了达到国标 ENF 级，还专门做了法国 A+ 的环保检测，您知道为什么吗？」
4. 焦点转移：「因为国标 ENF 检测的只有甲醛含量一种指标，而我们的因为需要出口欧洲的原因需要检测苯类、烃类等二十多种 VOC 有害物质，全部合格才能出口，所以我们选择的板材要求可能会比国内更严格一点，我们现在使用的是派阳山的 OSB 板为主，您听说过吗？」

### 六种风格话术体系

#### 1️⃣ 现代简约（80-120 平，强调性价比）

**拳头产品：** 恒温混色 PUR 封边 + 二百多种设计师花色 + 净味基材

**话术要点：**
- 一百多万的数码混合调色共翔封边机，恒温恒湿环境封边
- 花色是出口工厂拿的一手国外设计师花色，价格比国产福人还便宜
- 标配意大利比利斯小角度缓冲合页，连螺丝都是西班牙进口
- 如果房子只是临时过渡，不建议选我们（反向筛选）

#### 2️⃣ 意式极简（大面积，深色，注重质感）

**拳头产品：** 仿生水漆木皮 + 六面无缝亚克力

**话术要点：**
- 意式极简用免漆板做绝对做不出感觉
- 仿生木皮 399、六面无缝亚克力 399，别人成本都做不出来
- 就算不买我们，也一定要了解木皮和烤漆产品

#### 3️⃣ 中古风（木纹、骨骼线、藤编门）

**拳头产品：** 六轴加工中心 + 骨骼线门

**话术要点：**
- 花了近百万新上了六轴加工中心，把骨骼线门价格打下来 60%
- 骨骼线 899/980 一投影，40+ 花色可选
- 小工厂买不起加工中心，手工制作成本比我们的卖价还高

#### 4️⃣ 法式美式（造型门、水漆）

**拳头产品：** 美国宣伟水漆 + 实木基材

**话术要点：**
- 美国宣伟水漆，能喝的环保水性漆
- 三底四面工艺，拆开包没味道，当天装好当天住
- 38 种有害物质全部未检出
- 8 个门型、9 个颜色

#### 5️⃣ 门墙柜一体

**拳头产品：** 五门同色配套

**话术要点：**
- 二百多种花色全部打通门墙柜同色体系
- 同色隐形门 1880，锁子和合页不加钱

#### 6️⃣ 收纳类（橱柜/空间）

**拳头产品：** 收纳型橱柜

**话术要点：**
- 不依靠增加功能性五金，靠设计师巧思
- 4 米多厨房收纳能力比普通 10 米还多
- 相当于省下几平米房款
