## Excel 文件结构

**路径：** `C:\Users\LT-PC\Desktop\项目进度表.xlsx`

### 三个工作表

| 工作表 | 用途 | 说明 |
|--------|------|------|
| 参数调节 | 预警参数配置 | 腾哥直接改数字，条件格式自动关联 |
| Sheet2 | 流程定义 | 工序+责任人+时间规则+角色+订单要素 |
| 项目进度 | 实际数据记录 | 21列：客单号→售后时间+合同金额+投影面积 |

### 项目进度表列结构（21列）

A:客单号 → B:销售经理 → C:经销商 → D:终端地址 → E:设计师 → F:拆单 → G:接单时间 → H:报价时间 → I:回款时间 → J:采购时间 → K:图纸时间 → L:拆单完成 → M:采购到厂 → N:生产开始 → O:入库完成 → P:送货时间 → Q:安装完成时间 → R:验收时间 → S:售后时间 → T:合同金额 → U:投影面积

### 条件格式规则

每个时间列（H-S）各有 2 条规则：
- **黄色预警：** `AND(该列为空, TODAY()-接单时间 >= 参数调节!$B${row}, 参数调节!$B${row}>0)`
- **红色超期：** `AND(该列为空, TODAY()-接单时间 >= 参数调节!$C${row}, 参数调节!$B${row}>0)`

送货(P)和售后(S)的黄红天数设为 0 → 条件格式不触发，仅做记录。

## 飞书多维表格方案

### 核心能力

- **多角色填表：** 每个角色一个表单视图（销售/设计/仓库/安装），手机填完自动汇总
- **自动记时间：** 自动化规则 — 状态变更时自动写入当前时间
- **超期预警：** 条件格式变黄变红 + 飞书内推送
- **数据看板：** 仪表盘统计项目数、超期数、完成率
- **拍照上传：** 附件字段支持手机拍照

### 成本

- 免费版：¥0（多维表格无上限 + 表单视图 + 自动化规则 + 基础仪表盘）
- 标准版：¥10/人/月（10 人团队 ≈ 100元/月）

### 局限

- ❌ 员工需装飞书App（不能微信推送）
- ❌ 无法对接ERP自动拉订单
- ❌ 销售跟进时间线较难实现
- ❌ 数据在飞书服务器上
